Runtime Configuration Management

  • Create separate configs for dev, test, and prod environments
  • Use descriptive names: "OpenAI-Production-GPT4" vs "Config1"
  • Never hardcode API keys - use property placeholders
  • Test configs in Lab before using in production flows

Testing Operations

  • Start with simple operations to verify connectivity
  • Use the "Show" button on configs to verify all parameters
  • Check response times - slow responses may indicate config issues
  • Save successful parameter combinations for reuse

üîç Common Error Solutions

"401 Unauthorized" Error

Cause: Invalid or expired API key

Solution:

  1. Verify API key in your provider's dashboard
  2. Check for extra spaces or hidden characters
  3. Regenerate key if expired
  4. Update runtime configuration with new key

"Execution ID not found"

Cause: IDP execution not complete or incorrect ID

Solution:

  • Wait 5-10 seconds after submission before retrieving results
  • Double-check you copied the full executionId from response
  • Use "Get Execution Status" to check if processing is complete
